## 📌 프로젝트 개요

본 프로젝트는 **디지털 광고 클릭 및 전환 예측 모델 구축**을 통해 마케팅 의사결정을 데이터 기반으로 최적화하는 것을 목표로 합니다. 단순 모델 구현이 아니라, **마케팅 인사이트 도출 관점**에서 데이터 분석 → 모델링 → 전략적 시사점까지 연결하는 실습 프로젝트입니다.

---

## 🎯 프로젝트 목표 (Marketing Perspective)

디지털 마케팅에서 핵심 KPI는 다음과 같습니다:

* CTR (Click Through Rate)
* CVR (Conversion Rate)
* CAC (Customer Acquisition Cost)
* ROAS (Return on Ad Spend)

본 프로젝트는 **어떤 사용자와 어떤 광고가 전환 가능성이 높은지 예측**하여:

* 광고 예산 효율화
* 타겟 세그먼트 최적화
* 개인화 광고 전략 설계
  를 지원하는 것이 목적입니다.

---

## 📂 데이터 설명

`advertising.csv` 데이터셋은 디지털 광고 캠페인 로그 데이터를 기반으로 구성되어 있으며, 주요 컬럼은 다음과 같습니다:

| 컬럼                       | 설명            | 마케팅 의미      |
| ------------------------ | ------------- | ----------- |
| Daily Time Spent on Site | 사이트 체류 시간     | 관심도/몰입도 지표  |
| Age                      | 사용자 나이        | 세그먼트 타겟팅 기준 |
| Area Income              | 지역 소득 수준      | 구매력 프록시 변수  |
| Daily Internet Usage     | 일 평균 인터넷 사용량  | 디지털 친화도     |
| Ad Topic Line            | 광고 카피         | 크리에이티브 요소   |
| City / Country           | 지역            | 지리적 타겟팅     |
| Timestamp                | 광고 노출 시간      | 시간대별 성과 분석  |
| Clicked on Ad            | 전환 여부 (타겟 변수) | KPI 핵심 지표   |

## 🔍 분석 및 모델링 과정

### 1️⃣ 데이터 탐색 (EDA)

* 사용자 행동 변수와 클릭 전환 간 상관관계 분석
* 연령, 소득, 디지털 사용량 등 **세그먼트별 전환 패턴 분석**
* 시간대별 클릭률 트렌드 확인 (Dayparting 전략 인사이트)

### 2️⃣ 전처리

* 범주형 변수 인코딩 (지역, 광고 카피 등)
* 스케일링 및 결측치 처리
* Train/Test Split

### 3️⃣ 모델링

* Logistic Regression (baseline)
* Random Forest / Gradient Boosting (고성능 모델)
* 모델 성능 평가: Accuracy, ROC-AUC, Precision-Recall

## 📈 핵심 마케팅 인사이트

### ✅ 1. 사용자 몰입도는 전환의 핵심 드라이버

* 사이트 체류 시간이 길수록 클릭/전환 확률이 증가
* → 콘텐츠 품질 및 UX 개선이 전환율 상승에 직접 기여

### ✅ 2. 소득 수준과 전환은 비선형 관계

* 중간 소득층에서 전환율이 가장 높음 (가격 민감도 & 구매력 균형)
* → 고가 상품은 고소득층, 가성비 상품은 중소득층 타겟 전략 필요

### ✅ 3. 디지털 헤비 유저가 항상 전환되는 것은 아님

* 인터넷 사용량이 많을수록 광고 회피 성향 증가 가능성
* → 리타겟팅보다 신규 크리에이티브 전략 필요

### ✅ 4. 시간대별 전환 차이 존재

* 특정 시간대(예: 밤/점심)에 클릭률 상승 패턴 확인
* → **Dayparting 광고 예산 분배 전략** 적용 가능

### ✅ 5. 광고 카피(Ad Topic Line)의 영향

* 크리에이티브 메시지에 따라 CTR 편차 존재
* → A/B 테스트 기반 카피 최적화 필수

## 💡 실무 마케팅 적용 전략

### 🎯 타겟 세그먼트 전략

* High Time Spent + Mid Income 사용자 → Performance 캠페인 집중
* Low Engagement 사용자 → Awareness 캠페인 활용

### 💰 예산 최적화 전략

* 전환 확률 기반 입찰가 조정 (Bid Optimization)
* High CVR 세그먼트에 예산 집중 → CAC 감소

### 🧠 개인화 마케팅

* 예측 모델 결과를 활용해 Dynamic Creative Optimization (DCO)
* 사용자 특성별 맞춤 광고 메시지 제공
